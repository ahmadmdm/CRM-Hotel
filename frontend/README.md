# CrmHotel Frontend

Flutter application for the admin console and worker mobile flows.

## Supported SDK

- Flutter 3.41.x
- Dart 3.11.x
- Preferred SDK path: `../.tooling/flutter/bin/flutter`
- FVM pin file: `../.fvmrc`

## Optional FVM Setup

1. Install FVM.
2. Run `fvm use` from the workspace root.
3. Run `fvm flutter --version` and confirm it resolves to Flutter 3.41.9.

## Planned Run Commands

1. Run `../.tooling/flutter/bin/flutter pub get`.
2. Run `../.tooling/flutter/bin/flutter test`.
3. Run `../.tooling/flutter/bin/flutter run -d chrome` for the web admin panel or a mobile simulator for worker flows.
